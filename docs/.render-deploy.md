# Render.io Deployment

AI Fitness Agent can be deployed to Render.io with a single click using the included `render.yaml` configuration.

## Quick Deploy

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

Or follow the [Deployment Guide](DEPLOYMENT.md) for manual setup.

## Environment Variables

Set the following in Render Dashboard:
- `GEMINI_API_KEY` - Your Google Gemini API key

All other variables are auto-configured from `render.yaml`.
